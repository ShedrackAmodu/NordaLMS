Authentication

NexusLMS API uses OAuth 2.0 for authentication. You'll need to obtain an access token to make API requests.

Get Access Token
POST /oauth/token
Content-Type: application/json

{
  "grant_type": "client_credentials",
  "client_id": "your_client_id",
  "client_secret": "your_client_secret",
  "scope": "read write"
}

Getting Started

Quick Start Guide
  1. Obtain API credentials from your institution administrator
  2. Authenticate using OAuth 2.0 client credentials flow
  3. Make your first API request to verify connectivity
Note: API access is available on Professional and Enterprise plans.
